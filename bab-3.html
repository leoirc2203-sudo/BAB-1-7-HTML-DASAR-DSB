🔗 BAB 3: LINK & GAMBAR

"Biar websitenya bisa nyambung ke mana-mana dan keliatan kece pake gambar!"

🔗 1. LINK <a>

Link atau hyperlink pake tag <a> (anchor). Ini yang bikin website bisa "berselancar" ke halaman lain.

HTML 
<a href="https://google.com">Klik ke Google</a>

<a href="halaman2.html">Ke halaman lain di website sendiri</a>

<a href="#">Link gak kemana-mana (buat contoh)</a>
HASILNYA:

Klik ke Google (tapi di sini gak berfungsi ya, cuma contoh)

Ke halaman lain

Link gak kemana-mana

⬆️ Biasanya link berwarna biru dan ada garis bawah

📌 ATRIBUT PENTING:
  • href : alamat yang dituju (wajib ada!)
  • target="_blank" : buka di tab baru
  • title : teks tooltip saat hover
HTML (dengan atribut) 
<a href="https://google.com" target="_blank" title="Menuju Google">
    Buka Google di tab baru
</a>

🖼️ 2. GAMBAR <img>

Tag <img> buat nampilin gambar. Bedanya sama tag lain, img itu tag tunggal (gak perlu penutup).

HTML 
<img src="https://via.placeholder.com/300x200" alt="Gambar contoh">
HASILNYA:

⬆️ Gambar dari placeholder.com (gratis buat contoh)

📌 ATRIBUT PENTING:
  • src : sumber gambar (URL atau file lokal) - WAJIB!
  • alt : teks alternatif kalo gambar gagal load (penting buat aksesibilitas)
  • width & height : ukuran gambar

📁 3. GAMBAR DARI FILE LOKAL

Kalo gambar nyimpen di folder project sendiri, tinggal kasih pathnya.

HTML 
<!-- Kalo gambar satu folder sama file HTML -->
<img src="kucing.jpg" alt="Kucing lucu">

<!-- Kalo gambar di folder gambar/ -->
<img src="gambar/kucing.jpg" alt="Kucing lucu">

<!-- Kalo gambar di folder luar -->
<img src="../foto/kucing.jpg" alt="Kucing lucu">
💡 STRUKTUR FOLDER: 
belajar-html/
├── bab3.html
├── kucing.jpg           (satu folder)
├── gambar/
│   └── kucing.jpg       (di dalam folder)
└── foto/
    └── kucing.jpg       (folder lain)

🖱️ 4. GAMBAR SEBAGAI LINK

Gambar juga bisa jadi link tinggal di dalemin tag <a>.

HTML 
<a href="https://google.com">
    <img src="https://via.placeholder.com/200x100" alt="Klik gambar">
</a>
HASILNYA (coba klik gambarnya):

⬆️ Gambar ini bisa diklik (tapi di contoh gak kemana-mana)

⬇️ 5. LINK DALAM HALAMAN

Bisa bikin link yang loncat ke bagian tertentu dalam halaman yang sama. Pake atribut id.

HTML 
<!-- Bikin link ke bagian tertentu -->
<a href="#bab3">Loncat ke Bab 3</a>

<!-- Di bagian tujuan, kasih id -->
<h2 id="bab3">Bab 3: Link & Gambar</h2>
COBA KLIK:

🔝 Kembali ke atas (di contoh cuma alert)

📋 RINGKASAN BAB 3

Tag Fungsi Atribut Penting
<a> Membuat link href, target, title
<img> Menampilkan gambar src, alt, width, height
Tag tunggal: <img> (gak perlu penutup)

🚀 TUGAS PRAKTIK: Bikin Galeri Foto

Buat halaman HTML yang isinya:

  • Judul "Galeri Foto Keren"
  • 3 gambar (boleh pake gambar online/placeholder)
  • Setiap gambar bisa diklik dan membuka link ke sumber gambar
  • Link "Kembali ke atas" di bagian bawah
