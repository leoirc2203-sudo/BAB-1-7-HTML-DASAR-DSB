📘 BAB 1: HTML ITU APA SIH?

🏗️ Analogi: Rumah & Cor Beton

Bayangin lo mau bikin rumah. HTML itu kayak cor beton + kerangka rumah. Yang bikin rumah punya bentuk: ada tembok, ada jendela, ada pintu. Tanpa cor, rumah ga bakal berdiri!

Nah, kalo di website: HTML bikin struktur: ada judul, ada paragraf, ada gambar, ada tombol. CSS itu cat & keramiknya (biar bagus), JavaScript itu listrik & airnya (biar hidup).

💻 Struktur HTML Paling Dasar

<!DOCTYPE html>
<html>
<head>
    <title>Judul Website</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Ini website pertamaku.</p>
</body>
</html>

🔍 Penjelasan Kode:

  • <!DOCTYPE html> : Ngasih tau browser "Ini lho file HTML"
  • <html> : Tag pembuka, semua konten ada di dalem sini
  • <head> : Kepala website (buat settingan, judul, dll)
  • <title> : Judul yang muncul di tab browser
  • <body> : Badan website (isi yang keliatan)
  • <h1> : Heading/judul utama (paling gede)
  • <p> : Paragraf

🎮 COBA LANGSUNG!

▶️ Hasil dari kode di atas:

Halo Dunia!

Ini website pertamaku.

📌 CATATAN PENTING:

HTML itu bukan bahasa programming, tapi markup language. Maksudnya? HTML cuma buat nandain mana judul, mana paragraf, mana gambar. Ga ada logika matematika atau kondisi di HTML. Tapi HTML adalah DASAR dari semua website!

🛠️ Tools yang Lo Butuhin

  • Text Editor: Notepad, VS Code, atau Acode (kalo di HP)
  • Browser: Chrome, Firefox, Edge (buat liat hasilnya)
  • Semangat! Yang paling penting

📝 TUGAS PRAKTIK:

Buka text editor lo, ketik kode di atas, save sebagai pertamaku.html, terus buka pake browser. Kalo keliatan tulisannya "Halo Dunia!", SELAMAT! Lo baru aja bikin website pertama!
