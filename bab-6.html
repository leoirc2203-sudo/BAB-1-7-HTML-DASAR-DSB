🧱 BAB 6: DIV & LAYOUT

"Div itu kontainer ajaib buat ngatur tata letak website!"

📦 1. APA ITU DIV? <div>

<div> (division) adalah tag yang gunanya buat mengelompokkan elemen. Div itu kayak kontainer atau kotak yang bisa lo pake buat ngebungkus elemen HTML lain.

HTML 
<div style="background: #1e293b; padding: 20px;">
    <h2>Judul dalam div</h2>
    <p>Ini paragraf dalam div. Semua yang di sini jadi satu kelompok.</p>
</div>
HASILNYA:

Judul dalam div

Ini paragraf dalam div. Semua yang di sini jadi satu kelompok.

📌 KENAPA PAKE DIV?
  • Buat ngelompokin elemen yang related
  • Buat ngasih style CSS ke sekelompok elemen
  • Buat ngatur layout dengan Flexbox/Grid
  • Buat bikin struktur website yang rapi

🪆 2. DIV BERSARANG Div di dalam Div

Div bisa di dalam div lagi. Kayak kardus di dalam kardus.

HTML 
<div style="background: #2d3a4f; padding: 20px;">
    <h3>Div Luar</h3>
    <div style="background: #0f172a; padding: 15px; margin: 10px 0;">
        <p>Div dalam 1</p>
    </div>
    <div style="background: #0f172a; padding: 15px; margin: 10px 0;">
        <p>Div dalam 2</p>
    </div>
</div>
HASILNYA:

Div Luar

Div dalam 1

Div dalam 2

🏗️ 3. LAYOUT KLASIK Header, Content, Sidebar, Footer

Dengan div, lo bisa bikin layout website klasik yang punya header, sidebar, content, dan footer.

HTML 
<div class="header">HEADER</div>
<div class="sidebar">SIDEBAR</div>
<div class="content">CONTENT</div>
<div class="footer">FOOTER</div>
HASILNYA (pake CSS float):
HEADER
SIDEBAR
CONTENT UTAMA
FOOTER
💡 CATATAN: Layout di atas pake properti CSS float. Tapi sekarang lebih sering pake Flexbox atau Grid karena lebih gampang diatur!

📏 4. LAYOUT DENGAN FLEXBOX display: flex

Flexbox adalah cara modern buat ngatur layout. Cocok buat layout 1 dimensi (baris atau kolom).

HTML 
<div style="display: flex; gap: 10px;">
    <div style="flex: 1;">Kolom 1</div>
    <div style="flex: 1;">Kolom 2</div>
    <div style="flex: 1;">Kolom 3</div>
</div>
HASILNYA (3 kolom sama lebar):
Kolom 1
Kolom 2
Kolom 3
HTML (dengan ukuran berbeda) 
<div style="display: flex; gap: 10px;">
    <div style="flex: 1;">Sidebar (1)</div>
    <div style="flex: 3;">Content (3x lebih besar)</div>
</div>
HASILNYA (sidebar + content):
Sidebar
Content Utama

🔲 5. LAYOUT DENGAN CSS GRID display: grid

Grid adalah cara modern buat layout 2 dimensi (baris DAN kolom).

HTML 
<div style="display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px;">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
    <div>Item 4</div>
    <div>Item 5</div>
    <div>Item 6</div>
</div>
HASILNYA (grid 3 kolom):
Item 1
Item 2
Item 3
Item 4
Item 5
Item 6
HTML (layout kompleks) 
<div style="display: grid; grid-template-columns: 200px 1fr; gap: 10px;">
    <div style="background: #10b981;">Sidebar 200px</div>
    <div style="background: #3b82f6;">Content sisanya</div>
    <div style="grid-column: span 2; background: #8b5cf6;">Footer (gabung 2 kolom)</div>
</div>

🏷️ 6. SEMANTIC HTML Header, Nav, Main, Article, Section, Footer

HTML5 punya tag semantic yang lebih jelas maknanya daripada div. Tapi fungsinya sama: buat ngelompokin konten.

HTML 
<header>Ini header website</header>
<nav>Ini menu navigasi</nav>
<main>
    <article>
        <h2>Judul Artikel</h2>
        <p>Isi artikel...</p>
    </article>
    <aside>Ini sidebar (konten sampingan)</aside>
</main>
<footer>Ini footer</footer>
📌 PERBEDAAN DIV VS SEMANTIC:
  • <div> : generic container (gak ada makna khusus)
  • <header> : bagian kepala halaman
  • <nav> : navigasi
  • <main> : konten utama
  • <article> : artikel/konten independen
  • <section> : bagian dari halaman
  • <aside> : sidebar (konten sampingan)
  • <footer> : kaki halaman
💡 SARAN: Pake semantic HTML kalo memungkinkan! Lebih bagus buat SEO dan aksesibilitas. Tapi div tetep dipake kalo gak ada semantic tag yang cocok.

🌐 7. CONTOH LAYOUT WEBSITE LENGKAP

Ini contoh layout website sederhana pake div dan semantic HTML.

HTML 
<body>
    <header>
        <h1>Website Gue</h1>
        <nav>
            <a href="#">Home</a>
            <a href="#">About</a>
            <a href="#">Contact</a>
        </nav>
    </header>
    
    <main style="display: flex; gap: 20px;">
        <article style="flex: 3;">
            <h2>Judul Artikel</h2>
            <p>Isi artikel...</p>
        </article>
        
        <aside style="flex: 1;">
            <h3>Sidebar</h3>
            <p>Konten sampingan...</p>
        </aside>
    </main>
    
    <footer>
        <p>© 2026 Website Gue</p>
    </footer>
</body>
HASILNYA (kurang lebih):

Website Gue

Home About Contact

Judul Artikel

Ini adalah contoh artikel. Lo bisa nulis apa aja di sini. Artikel ini pake tag <article> yang artinya konten independen.

Sidebar

Ini sidebar, biasanya buat link, iklan, atau info tambahan.

© 2026 Website Gue

📋 RINGKASAN BAB 6

Tag Fungsi
<div> Kontainer umum (paling sering dipake)
<header> Kepala halaman
<nav> Navigasi
<main> Konten utama
<article> Artikel/konten independen
<section> Bagian dari halaman
<aside> Sidebar/konten sampingan
<footer> Kaki halaman
💡 TEKNIK LAYOUT MODERN:
  • Flexbox : buat layout 1 dimensi (baris ATAU kolom)
  • CSS Grid : buat layout 2 dimensi (baris DAN kolom)
  • Float : cara lama, udah jarang dipake

🚀 TUGAS PRAKTIK: Bikin Layout Blog

Buat layout blog sederhana dengan struktur:

  • Header (judul blog + navigasi)
  • Main content dengan 2 kolom:
    • Kolom kiri: daftar artikel (pake artikel)
    • Kolom kanan: sidebar (tentang penulis)
  • Footer
  • Pake semantic HTML dan Flexbox/Grid
