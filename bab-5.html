📝 BAB 5: FORM (INPUT DATA)

"Bikin form buat login, daftar, atau ngirim data ke server!"

📋 1. STRUKTUR DASAR FORM <form>

Tag <form> adalah wadah buat semua elemen input.

HTML 
<form action="/proses.php" method="post">
    <!-- Semua elemen form di sini -->
    <input type="text" name="nama">
    <button type="submit">Kirim</button>
</form>
📌 ATRIBUT PENTING:
  • action : alamat tujuan data dikirim
  • method : cara kirim data (GET atau POST)
  • GET : data keliatan di URL (buat pencarian)
  • POST : data dikirim diam-diam (buat login/daftar)

✏️ 2. INPUT TEXT <input type="text">

Buat input teks pendek kayak nama, email, dll.

HTML 
<form>
    <label for="nama">Nama Lengkap:</label>
    <input type="text" id="nama" name="nama" placeholder="Masukkan nama">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="contoh@email.com">
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" placeholder="******">
</form>
HASILNYA:
💡 TYPE LAINNYA:
  • type="email" : validasi email
  • type="password" : karakter disembunyiin
  • type="number" : cuma angka
  • type="tel" : buat nomor telepon
  • type="url" : buat alamat website

📄 3. TEXTAREA <textarea>

Buat input teks panjang kayak komentar, pesan, deskripsi.

HTML 
<form>
    <label for="pesan">Pesan:</label>
    <textarea id="pesan" name="pesan" rows="5" cols="40" 
        placeholder="Tulis pesan lo di sini..."></textarea>
</form>

🔘 4. RADIO BUTTON Pilih Satu

Radio button buat pilihan yang cuma bisa milih SATU.

HTML 
<form>
    <p>Jenis Kelamin:</p>
    <input type="radio" id="pria" name="gender" value="pria">
    <label for="pria">Pria</label><br>
    
    <input type="radio" id="wanita" name="gender" value="wanita">
    <label for="wanita">Wanita</label><br>
    
    <input type="radio" id="lain" name="gender" value="lain">
    <label for="lain">Lainnya</label>
</form>
HASILNYA:

Jenis Kelamin:

⚠️ PENTING: Radio button harus punya name yang SAMA biar cuma bisa milih satu!

✅ 5. CHECKBOX Pilih Banyak

Checkbox buat pilihan yang bisa milih LEBIH DARI SATU.

HTML 
<form>
    <p>Hobi:</p>
    <input type="checkbox" id="coding" name="hobi[]" value="coding">
    <label for="coding">Coding</label><br>
    
    <input type="checkbox" id="game" name="hobi[]" value="game">
    <label for="game">Game</label><br>
    
    <input type="checkbox" id="musik" name="hobi[]" value="musik">
    <label for="musik">Musik</label>
</form>

📋 6. SELECT DROPDOWN <select>

Dropdown buat pilihan yang irit tempat.

HTML 
<form>
    <label for="kota">Pilih Kota:</label>
    <select id="kota" name="kota">
        <option value="">-- Pilih Kota --</option>
        <option value="jakarta">Jakarta</option>
        <option value="bandung">Bandung</option>
        <option value="surabaya">Surabaya</option>
        <option value="medan">Medan</option>
    </select>
</form>
HASILNYA:

🖱️ 7. BUTTON <button>

Tombol buat kirim form atau reset.

HTML 
<form>
    <button type="submit">Kirim Data</button>
    <button type="reset">Reset Form</button>
    <button type="button" onclick="alert('Tombol diklik!')">
        Tombol Biasa
    </button>
</form>
HASILNYA:
📌 TYPE BUTTON:
  • type="submit" : kirim form ke server
  • type="reset" : reset semua input ke default
  • type="button" : tombol biasa (biasa pake JavaScript)

📋 RINGKASAN BAB 5

Tag/Type Fungsi
<form> Container form
<input type="text"> Input teks pendek
<input type="email"> Input email (validasi otomatis)
<input type="password"> Input password (bullets)
<input type="number"> Input angka
<input type="radio"> Pilih satu (name harus sama)
<input type="checkbox"> Pilih banyak
<textarea> Input teks panjang
<select> & <option> Dropdown pilihan
<button> Tombol
<label> Label buat input (aksesibilitas)

🚀 TUGAS PRAKTIK: Bikin Form Pendaftaran

Buat form pendaftaran dengan elemen:

  • Nama (text)
  • Email (email)
  • Password (password)
  • Jenis Kelamin (radio)
  • Hobi (checkbox - minimal 3 pilihan)
  • Kota (select dropdown)
  • Pesan (textarea)
  • Tombol Daftar (submit)
