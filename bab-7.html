🎯 TUGAS AKHIR: BIKIN WEBSITE SENDIRI!

📋 SPESIFIKASI MINIMAL:

  • ✅ Pake struktur HTML yang benar (DOCTYPE, html, head, body)
  • ✅ Minimal 5 jenis tag berbeda (heading, paragraf, link, gambar, list, dll)
  • ✅ Ada navigasi (menu)
  • ✅ Ada gambar (bisa dari internet atau file lokal)
  • ✅ Ada form (minimal input nama dan email)
  • ✅ Pake div buat layout (header, content, footer)
  • ✅ Ada CSS styling (warna, font, layout)

💡 IDE TAMBAHAN:

  • ✨ Responsive design (bisa dibuka di HP)
  • ✨ Animasi sederhana (hover effects)
  • ✨ Flexbox atau Grid buat layout
  • ✨ Semantic HTML (header, nav, main, footer)
  • ✨ List, tabel, atau form yang kompleks
