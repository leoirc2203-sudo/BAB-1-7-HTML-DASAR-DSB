📋 BAB 4: LIST & TABEL

"Biar data di website lo rapi dan gak berantakan!"

🔘 1. UNORDERED LIST <ul> & <li>

Unordered list buat daftar yang urutannya gak penting. Biasanya pake bullet (lingkaran, kotak, dll).

HTML 
<ul>
    <li>Nasi goreng</li>
    <li>Mie ayam</li>
    <li>Bakso</li>
    <li>Sate</li>
</ul>
HASILNYA:
  • Nasi goreng
  • Mie ayam
  • Bakso
  • Sate
💡 ATRIBUT type pada <ul>:
  • type="disc" : bullet bentuk lingkaran (default)
  • type="circle" : bullet lingkaran bolong
  • type="square" : bullet kotak

🔢 2. ORDERED LIST <ol> & <li>

Ordered list buat daftar yang urutannya penting. Pake angka, huruf, atau romawi.

HTML 
<ol>
    <li>Bangun tidur</li>
    <li>Sarapan</li>
    <li>Belajar HTML</li>
    <li>Tidur lagi</li>
</ol>
HASILNYA:
  1. Bangun tidur
  2. Sarapan
  3. Belajar HTML
  4. Tidur lagi
📌 ATRIBUT <ol>:
  • type="1" : angka (default)
  • type="A" : huruf kapital (A, B, C)
  • type="a" : huruf kecil (a, b, c)
  • type="I" : romawi kapital (I, II, III)
  • type="i" : romawi kecil (i, ii, iii)
  • start="5" : mulai dari angka 5
HTML (contoh type) 
<ol type="A">
    <li>Pilihan A</li>
    <li>Pilihan B</li>
    <li>Pilihan C</li>
</ol>

<ol type="I" start="3">
    <li>Mulai dari III</li>
    <li>Ini IV</li>
    <li>Ini V</li>
</ol>
HASILNYA:
  1. Pilihan A
  2. Pilihan B
  3. Pilihan C
  1. Mulai dari III
  2. Ini IV
  3. Ini V

🪆 3. NESTED LIST List Bersarang

Bisa bikin list di dalam list. Kayak daftar isi buku.

HTML 
<ul>
    <li>Makanan
        <ul>
            <li>Nasi goreng</li>
            <li>Mie ayam</li>
        </ul>
    </li>
    <li>Minuman
        <ul>
            <li>Es teh</li>
            <li>Kopi</li>
        </ul>
    </li>
</ul>
HASILNYA:
  • Makanan
    • Nasi goreng
    • Mie ayam
  • Minuman
    • Es teh
    • Kopi

📊 4. TABEL DASAR <table>, <tr>, <td>

Tabel buat nampilin data dalam baris dan kolom.

HTML 
<table border="1">
    <tr>
        <td>Baris 1, Kolom 1</td>
        <td>Baris 1, Kolom 2</td>
    </tr>
    <tr>
        <td>Baris 2, Kolom 1</td>
        <td>Baris 2, Kolom 2</td>
    </tr>
</table>
HASILNYA:
Baris 1, Kolom 1 Baris 1, Kolom 2
Baris 2, Kolom 1 Baris 2, Kolom 2
📌 PENJELASAN:
  • <table> : bikin tabel
  • <tr> : table row (baris)
  • <td> : table data (sel/kotak)
  • border="1" : kasih garis tepi

🏷️ 5. TABEL DENGAN HEADER <th>

Pake <th> (table header) buat judul kolom. Biasanya teksnya tebal.

HTML 
<table border="1">
    <tr>
        <th>Nama</th>
        <th>Umur</th>
        <th>Kota</th>
    </tr>
    <tr>
        <td>Budi</td>
        <td>20</td>
        <td>Jakarta</td>
    </tr>
    <tr>
        <td>Ani</td>
        <td>19</td>
        <td>Bandung</td>
    </tr>
</table>
HASILNYA:
Nama Umur Kota
Budi 20 Jakarta
Ani 19 Bandung

🔄 6. GABUNGIN SEL colspan & rowspan

Buat gabungin beberapa kolom (colspan) atau baris (rowspan).

HTML 
<table border="1">
    <tr>
        <th colspan="2">Data Diri</th>
        <th>Umur</th>
    </tr>
    <tr>
        <td>Budi</td>
        <td>Jakarta</td>
        <td>20</td>
    </tr>
    <tr>
        <td>Ani</td>
        <td>Bandung</td>
        <td>19</td>
    </tr>
</table>
HASILNYA (colspan):
Data Diri Umur
Budi Jakarta 20
Ani Bandung 19
HTML (rowspan) 
<table border="1">
    <tr>
        <th rowspan="2">Nama</th>
        <th colspan="2">Detail</th>
    </tr>
    <tr>
        <th>Kota</th>
        <th>Umur</th>
    </tr>
    <tr>
        <td>Budi</td>
        <td>Jakarta</td>
        <td>20</td>
    </tr>
</table>
HASILNYA (rowspan + colspan):
Nama Detail
Kota Umur
Budi Jakarta 20

📋 RINGKASAN BAB 4

Tag Fungsi Atribut Penting
<ul> List tidak berurutan type (disc, circle, square)
<ol> List berurutan type (1, A, a, I, i), start
<li> Item list value (khusus <ol>)
<table> Membuat tabel border, cellpadding, cellspacing
<tr> Baris tabel -
<td> Sel data colspan, rowspan
<th> Sel header colspan, rowspan

🚀 TUGAS PRAKTIK: Bikin Jadwal Pelajaran

Buat tabel jadwal pelajaran dengan ketentuan:

  • Kolom: Hari, Jam 1, Jam 2, Jam 3
  • Baris: Senin, Selasa, Rabu, Kamis, Jumat
  • Gabungin sel Jumat untuk "Jumat Agung" (colspan)
  • Kasih header tabel yang rapi

Hari Jam 1 (07:30) Jam 2 (09:15) Jam 3 (10:45)
Senin Matematika IPA IPS
Selasa Bahasa Indo Bahasa Inggris Olahraga
Rabu Agama PKN Seni Budaya
Kamis Matematika IPA IPS
Jumat 📿 Jumat Agung
